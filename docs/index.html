O Projekcie

Pot-F Desktop to polski fork aplikacji pot-app/pot-desktop autorstwa Pylogmon. Projekt dostosowuje aplikację do polskich użytkowników oraz systemów z rodziny Linux (w szczególności Fedora 43).

🔄 Tłumaczenie tekstu

Zaznacz tekst i naciśnij skrót klawiszowy lub wpisz ręcznie. Wielowątkowe tłumaczenie z wielu usług jednocześnie.

📸 OCR zrzutu ekranu

Zrób zrzut ekranu i zamień obraz na tekst. Wsparcie dla Tesseract OCR na Linux.

📋 Monitorowanie schowka

Automatyczne tłumaczenie tekstu skopiowanego do schowka w tle.

🔊 Synteza mowy (TTS)

Zamień przetłumaczony tekst na mowę. Wsparcie dla Lingva i wtyczek zewnętrznych.

📚 Eksport do słowników

Integracja z Anki i innymi słownikami do nauki słówek.

🔌 System wtyczek

Rozszerzaj funkcjonalność o nowe usługi tłumaczenia, OCR i TTS.

Instalacja

Fedora 43 (RPM)

  1. Pobierz paczkę z GitHub Releases
  2. Zainstaluj: 
    sudo dnf install ./pot-f-desktop-0.1.0-1.x86_64.rpm
  3. Zainstaluj wtyczki: 
    sudo dnf install ./pot-f-desktop-plugins-0.1.0-1.fc43.noarch.rpm

Arch Linux / Manjaro

Pobierz paczkę DEB i konwertuj na Arch.pkg.tar.zst:

sudo debtap ./pot-f-desktop_0.1.0_amd64.deb
sudo pacman -U ./pot-f-desktop-0.1.0-1-x86_64.pkg.tar.zst

Lub zbuduj z kodu źródłowego (patrz Budowa z kodu).

AppImage (uniwersalna paczka dla Linux)

Pobierz plik Pot-F-0.1.0-x86_64.AppImage ze strony GitHub Releases

chmod +x Pot-F-0.1.0-x86_64.AppImage
./Pot-F-0.1.0-x86_64.AppImage

Obsługa Wayland

Wsparcie dla Wayland zależy od dystrybucji i środowiska. Poniżej typowe rozwiązania:

Skróty klawiszowe nie działają

Tauri nie obsługuje skrótów na Wayland. Skonfiguruj skróty systemowe z wywołaniem przez curl:

curl "127.0.0.1:60828/selection_translate"

Zrzut ekranu nie działa (Hyprland)

Użyj grim + slurp:

bind = ALT, X, exec, grim -g "$(slurp)" \
  ~/.cache/com.pot-f-desktop.desktop/pot_screenshot_cut.png \
  && curl "127.0.0.1:60828/ocr_recognize?screenshot=false"

Okno pod kursorem (Hyprland windowrulev2)

windowrulev2 = float, class:(pot-f-desktop), \
  title:(Translator|OCR|PopClip|Screenshot Translate)
windowrulev2 = move cursor 0 0, class:(pot-f-desktop), \
  title:(Translator|PopClip|Screenshot Translate)

Wywołanie zewnętrzne (HTTP API)

Pot-F Desktop udostępnia interfejs HTTP na porcie 60828 (można zmienić w ustawieniach).

Endpoint Metoda Opis
/POSTTłumacz tekst (body: tekst)
/configGETOtwórz ustawienia
/translatePOSTTłumacz tekst
/selection_translateGETTłumacz zaznaczenie
/input_translateGETTłumacz z wprowadzania
/ocr_recognizeGETOCR zrzutu ekranu
/ocr_translateGETTłumaczenie zrzutu ekranu

Przykład

curl "127.0.0.1:60828/selection_translate"

Użycie z Flameshot (Linux)

rm ~/.cache/com.pot-f-desktop.desktop/pot_screenshot_cut.png \
  && flameshot gui -s -p ~/.cache/com.pot-f-desktop.desktop/pot_screenshot_cut.png \
  && curl "127.0.0.1:60828/ocr_recognize?screenshot=false"

Budowa z kodu

Wymagania

  • Node.js >= 18.0.0
  • pnpm >= 8.5.0
  • Rust >= 1.80.0

Kroki

  1. Sklonuj repozytorium: 
    git clone https://github.com/frs777/pot-f-desktop.git
cd pot-f-desktop
  2. Zainstaluj zależności: 
    pnpm install
  3. Tryb deweloperski: 
    pnpm tauri dev
  4. Budowa paczki: 
    pnpm tauri build --bundles rpm
