About the Project

Pot-F Desktop is a Polish fork of pot-app/pot-desktop by Pylogmon. This project adapts the application for Polish users and Linux systems (especially Fedora 43).

🔄 Text Translation

Select text and press keyboard shortcut or type manually. Multi-threaded translation from multiple services simultaneously.

📸 Screenshot OCR

Take a screenshot and convert image to text. Support for Tesseract OCR on Linux.

📋 Clipboard Monitoring

Automatic translation of text copied to clipboard in the background.

🔊 Text-to-Speech (TTS)

Convert translated text to speech. Support for Lingva and external plugins.

📚 Dictionary Export

Integration with Anki and other dictionaries for vocabulary learning.

🔌 Plugin System

Extend functionality with new translation, OCR and TTS services.

Features

  • ✅ Multi-threaded translation from multiple services simultaneously
  • ✅ Text recognition (OCR) from multiple services
  • ✅ Text-to-Speech (TTS)
  • ✅ Dictionary export (Anki, etc.)
  • ✅ External invocation via HTTP API
  • ✅ Plugin system for extensibility
  • ✅ Support for all PC platforms (via original project)
  • ✅ Multilingual (Polish interface by default)

Installation

Fedora 43 (RPM)

  1. Download the package from GitHub Releases
  2. Install: 
    sudo dnf install ./pot-f-desktop-0.1.0-1.x86_64.rpm
  3. Install plugins: 
    sudo dnf install ./pot-f-desktop-plugins-0.1.0-1.fc43.noarch.rpm

Arch Linux / Manjaro

Download DEB package and convert to Arch.pkg.tar.zst:

sudo debtap ./pot-f-desktop_0.1.0_amd64.deb
sudo pacman -U ./pot-f-desktop-0.1.0-1-x86_64.pkg.tar.zst

Or build from source (see Building from source).

AppImage (universal Linux package)

Download Pot-F-0.1.0-x86_64.AppImage from GitHub Releases

chmod +x Pot-F-0.1.0-x86_64.AppImage
./Pot-F-0.1.0-x86_64.AppImage

Wayland Support

Wayland support depends on distribution and desktop environment. Below are typical solutions:

Keyboard shortcuts don't work

Tauri doesn't support shortcuts on Wayland. Configure system shortcuts with curl invocation:

curl "127.0.0.1:60828/selection_translate"

Screenshot doesn't work (Hyprland)

Use grim + slurp:

bind = ALT, X, exec, grim -g "$(slurp)" \
  ~/.cache/com.pot-f-desktop.desktop/pot_screenshot_cut.png \
  && curl "127.0.0.1:60828/ocr_recognize?screenshot=false"

Window under cursor (Hyprland windowrulev2)

windowrulev2 = float, class:(pot-f-desktop), \
  title:(Translator|OCR|PopClip|Screenshot Translate)
windowrulev2 = move cursor 0 0, class:(pot-f-desktop), \
  title:(Translator|PopClip|Screenshot Translate)

External Invocation (HTTP API)

Pot-F Desktop exposes an HTTP interface on port 60828 (can be changed in settings).

Endpoint Method Description
/POSTTranslate text (body: text)
/configGETOpen settings
/translatePOSTTranslate text
/selection_translateGETTranslate selection
/input_translateGETInput translation
/ocr_recognizeGETScreenshot OCR
/ocr_translateGETScreenshot translation

Example

curl "127.0.0.1:60828/selection_translate"

Usage with Flameshot (Linux)

rm ~/.cache/com.pot-f-desktop.desktop/pot_screenshot_cut.png \
  && flameshot gui -s -p ~/.cache/com.pot-f-desktop.desktop/pot_screenshot_cut.png \
  && curl "127.0.0.1:60828/ocr_recognize?screenshot=false"

Building from Source

Requirements

  • Node.js >= 18.0.0
  • pnpm >= 8.5.0
  • Rust >= 1.80.0

Steps

  1. Clone the repository: 
    git clone https://github.com/frs777/pot-f-desktop.git
cd pot-f-desktop
  2. Install dependencies: 
    pnpm install
  3. Development mode: 
    pnpm tauri dev
  4. Build package: 
    pnpm tauri build --bundles rpm
